
# EZNet V2 (InDev)

🚀 EZNet a DOP based Roblox networking module, which has support for both OOP and DOD typing paragdims.

🐛 This is a Alpha major bugs are to be expected.

🛠️ More EZNet API is to be added.

📝 EZNet V2 will that a completly different approch from EZNet V1 and will not be based on the previous Rmv4 or Rmv4.1 nor Rmv4.1Super architecture but instead be more performance focused and fixing major flaws of the previous Rmv4 and Rmv4.1 flaws and the few Rmv3 ones that sneaked back in, on another note, EZNet V2 will finally break free of the chains called GoodSignal, and migrate to a DOD native signal system instead.
